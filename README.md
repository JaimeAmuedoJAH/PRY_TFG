# Seguimiento de Ligas Andaluces de Tenis de Mesa

## 📌 Descripción
Este proyecto es la **aplicación móvil de seguimiento de ligas andaluzas de tenis de mesa**, desarrollada como Trabajo de Fin de Grado Superior en **Desarrollo de Aplicaciones Multiplataforma (DAM)**.

La app permite a jugadores, entrenadores y aficionados:

- Visualizar partidos programados y resultados.
- Consultar clasificaciones de las ligas.
- Acceder a las actas de los partidos.
- Gestionar información de jugadores y equipos.

La aplicación está desarrollada en **Android Studio** con **Java**, y utiliza **Firebase** para la gestión de base de datos y autenticación en tiempo real.

---

## 🛠 Tecnologías y Herramientas

- **Lenguaje:** Java
- **IDE:** Android Studio
- **Backend / BBDD:** Firebase Realtime Database
- **Autenticación y seguridad:** Firebase Authentication
- **Almacenamiento de actas y archivos:** Firebase Storage
- **Control de versiones:** Git / GitHub
- **Gestión de dependencias:** Gradle

---

## ⚡ Características Principales

1. **Inicio de sesión y autenticación**
    - Usuarios registrados y acceso mediante email/password.
2. **Visualización de partidos**
    - Lista de partidos por jornada y liga.
    - Información de fecha, hora y resultado.
3. **Clasificaciones**
    - Tabla de posiciones de cada liga.
    - Actualización automática tras cada partido.
4. **Actas de partidos**
    - Consulta y descarga de actas oficiales.
5. **Gestión de ligas y equipos**
    - Posibilidad de agregar, modificar o eliminar equipos y jugadores (solo administradores).
