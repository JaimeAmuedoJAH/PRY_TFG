# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

### 1.0.1 (2025-05-26)

## [1.3.1] - 2025-05-26
### Mejorado
- Adaptación de layouts para compatibilidad con diferentes tamaños de dispositivos.
- Revisión general de clases para mejorar el uso de recursos de tipo `string`.

## [1.3.0] - 2025-05-18
### Refactorizado
- Refactorización de clases para pruebas unitarias: `AdaptadorClasificación` y `AdaptadorRanking`.
- Nuevas clases para manejar la lógica de negocio, pruebas unitarias de clasificación y ranking.

### Agregado
- Verificación por correo electrónico en el registro como requisito para iniciar sesión.
- Requisitos en la contraseña: mayúscula, número y carácter especial.

## [1.2.2] - 2025-05-23
### Modificado
- En `ActaActivity`, los campos de texto se vuelven no editables una vez que el partido ha sido jugado.

## [1.2.1] - 2025-05-22
### Completado
- Finalización de `ActaActivity`.
- Mejora en el diseño de los layouts de `Perfil` y `verPartidoActivity`.

## [1.2.0] - 2025-05-21
### Refactorizado
- Avance en `ActaActivity` con layout completo y nuevas funcionalidades.
- (Pendiente) Modificaciones en la base de datos al guardar el acta.

## [1.1.0] - 2025-05-14
### Mejorado
- Actualización de `EditarPerfilActivity` con vistas configuradas según el tipo de usuario (entrenador o jugador).
